# Projet PWA - FrontEnd

Petit projet de Progressive Web App (PWA) réalisé pour le cours de dév FrontEnd.
L'objectif était de créer une application web installable avec des fonctionnalités temps réel via socket.io.

# Stack technique

- Next.js
- TypeScript
- Tailwind CSS
- Socket.io

## Installation et lancement

1. Installer les paquets :

npm install
npm run dev

## Fonctionnalités principales

- PWA : Application installable (fichier `manifest.ts`).
- Rooms : Système de salons avec communication temps réel.
- Notifications : Page de gestion des alertes.
- Galerie : Affichage d'images.
